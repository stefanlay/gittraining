Push new Patch Set
------------------
A Common Scenario:

Author of Patch Set 1 is not available and somebody
else needs to continue and provide Patch Set 2

[role="incremental"]
[[push-new-patch-set-list]]
--

* `fetch`
* create a new branch based on the fetched patch set 1
* fix the issue
* `commit --amend`
* `push`
--
